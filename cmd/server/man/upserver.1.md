# NAME
dus \- DUC (Dockyard Updater Server)

# SYNOPSIS
**dus** [OPTIONS] COMMAND [arg...]

**dus** [--help|-v|--version]

# DESCRIPTION
dus is the "Dockyard Updater Server" working with Dockyard Server.


# OPTIONS
**--help**
  Print usage statement

**-v**, **--version**
  Print version information.

# COMMANDS
**web**
  Start Dockyard Updater Server.
